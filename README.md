# Customer_Segmentation_Project
Customer segmentation project using K-Means clustering in Python to analyze customer behavior and spending patterns.
## Objective
To segment customers based on income and spending behavior using K-Means clustering.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Algorithm
K-Means Clustering

## Output
Customers were grouped into different segments based on purchasing behavior.

## Conclusion
Customer segmentation helps businesses improve marketing strategies and understand customer preferences.
